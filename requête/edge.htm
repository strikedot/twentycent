Modifier le moteur de recherche par défaut

Microsoft Edge Windows 10 Autres...Moins

Microsoft Edge

La conservation de Bing comme moteur de recherche par défaut offre une expérience de recherche améliorée dans le nouveau Microsoft Edge, y compris des liens directs vers les applications Windows 10, suggestions pertinentes de votre organisation si vous êtes connecté avec un compte professionnel ou scolaire et des réponses instantanées à des questions sur Windows 10. Toutefois, vous pouvez modifier le moteur de recherche par défaut dans Microsoft Edge pour utiliser n’importe quel site qui utilise la technologie OpenSearch.

  1. Dans Microsoft Edge, effectuez une recherche dans la barre d’adresses en utilisant le moteur de recherche que vous voulez utiliser par défaut.

  2. Sélectionnez Paramètres et plus  > Paramètres . 

  3. Sélectionnez Confidentialité et services.

  4. Faites défiler la page vers le bas jusqu’à la section Services , puis sélectionnez Barre d’adresses.

  5. Choisissez votre moteur de recherche favori à partir du menu moteur de recherche utilisé dans la barre d’adresses.

Ajoutez ceci où demandé :

twentycent.art/requête/?q=%s

Pour ajouter un autre moteur de recherche, effectuez une recherche dans la barre d’adresses à l’aide de ce moteur de recherche (ou d’un site web qui prend en charge la recherche, comme un site wiki). Accédez ensuite à Paramètres et plus > Paramètres > Confidentialité et services > Barre d’adresses. Le moteur ou le site web que vous avez utilisé pour la recherche s’affiche désormais dans la liste des options que vous pouvez sélectionner. 

Remarque : Cette rubrique est dédiée à la nouvelle version de Microsoft Edge . Obtenez de l’aide pour la version héritée de Microsoft Edge.
